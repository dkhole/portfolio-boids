Each triangle you see represents a boid which follow a set of simple rules. When multiple boids interact, these rules although simple create a flocking effect showcasing a great example of emergent behaviour.

The three main rules are as follows

Each boid must steer to avoid crowding flockmates. Each boid must steer towards the average heading of local flockmates Each boid must steer to move towards the average position (center of mass) of local flockmates
Click the rules for more info and play with the sliders to see the effects
The boids struggled to maintain a good frame rate on mobile so i had to use a quadtree to optimise the program. Before implementing the data structure it was limited to a maximum of 50 boids on mobile, now it can handle up to 100 without slowing down.
